# Проект. Нейросеть для автодополнения текстов

## Задача
Создать нейросеть, которая на основе начала фразы предсказывает её продолжение.
Реализовать многократное предсказание следующего токена, пока не будет достигнут токен окончания последовательности либо лимит длины.

## Данные:
Датасет с короткими постами `sentiment140` (1.6 млн. текстов).

## Результаты и выводы по проекту

В ходе выполнения использованы и проанализированы две модели:
модель на основе LSTM и предобученная модель `distilgpt2` из `Transformers`.
Оценка качества проведена по результатам генерации последней четверти подаваемого на вход текста по метрике `ROUGE1`.

### Модель на основе рекуррентной нейронной сети
Значение метрики `ROUGE` на валидации: **0.0959**

### Модель на основе трансформера
Значение метрики `ROUGE` на валидации: *0.0750*


На валидации модель на основе LSTM немного лучше по метрике `ROUGE` (на 2 процентных пункта).
Данное значение **0.0959** говорит о том, что примерно 9.5 % слов из эталонного текста встретились в сгенерированном тексте.
Обе модели генерируют более или менее осмысленные продолжения.
